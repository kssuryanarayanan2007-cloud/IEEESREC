# AECTSD 2026 Conference Website

A clone of the AECTSD 2026 conference website built with React, Vite, TypeScript, Tailwind CSS, and shadcn/ui.

## Pages

- **Home** (`/`)
- **About Us** (`/about`)
- **Committee** — `/committee/organizing`, `/committee/steering`, `/committee/advisory`
- **Authors** (`/author`)
- **Registration** (`/register`)
- **Call for Papers** (`/call`)
- **Speakers** (`/speakers`)
- **Venue** (`/venue`)
- **Contact** (`/contact`)

## Run locally

```bash
npm install
npm run dev      # http://localhost:8080
npm run build
npm run preview
```

## Tech

React 18 · Vite 5 · TypeScript · Tailwind CSS v3 · shadcn/ui · React Router v6
